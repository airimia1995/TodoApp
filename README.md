# TodoApp

Todo list app using NextJs (React) and NestJs (NodeJs)

------------------
Start database

> cd fastStart<br />
> docker-compose up -d<br />

------------------
Starting client

> cd client<br />
> yarn<br />
> yarn dev<br />

------------------
Starting server

> cd server<br />
> yarn<br />
> yarn start<br />


--------------------

!!! the app has a small flickering because the styled components are not rendering properly in nextJs 13 !!!
